File operations

```shell title="Usage"
jj file <COMMAND>
```

### Subcommands

- [`annotate`](./jj/file/annotate) — Show the source change for each line of the target file
- [`chmod`](./jj/file/chmod) — Sets or removes the executable bit for paths in the repo
- [`list`](./jj/file/list) — List files in a revision
- [`show`](./jj/file/show) — Print contents of files in a revision
- [`track`](./jj/file/track) — Start tracking specified paths in the working copy
- [`untrack`](./jj/file/untrack) — Stop tracking specified paths in the working copy
