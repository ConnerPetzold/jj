Manage which paths from the working-copy commit are present in the working copy

```shell title="Usage"
jj sparse <COMMAND>
```

### Subcommands

- [`edit`](./jj/sparse/edit) — Start an editor to update the patterns that are present in the working copy
- [`list`](./jj/sparse/list) — List the patterns that are currently present in the working copy
- [`reset`](./jj/sparse/reset) — Reset the patterns to include all files in the working copy
- [`set`](./jj/sparse/set) — Update the patterns that are present in the working copy
