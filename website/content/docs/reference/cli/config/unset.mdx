title: jj config unset
isCommand: true
---

Update a config file to unset the given option

```shell title="Usage"
jj config unset <--user|--repo> <NAME>
```

**Command Alias:** `u`

### Arguments

- `<NAME>`

### Options

- `--user` — Target the user-level config
- `--repo` — Target the repo-level config
