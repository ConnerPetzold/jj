Infrequently used commands such as for generating shell completions

```shell title="Usage"
jj util <COMMAND>
```

### Subcommands

- [`completion`](./jj/util/completion) — Print a command-line-completion script
- [`config-schema`](./jj/util/config-schema) — Print the JSON schema for the jj TOML config format
- [`exec`](./jj/util/exec) — Execute an external command via jj
- [`gc`](./jj/util/gc) — Run backend-dependent garbage collection
- [`install-man-pages`](./jj/util/install-man-pages) — Install Jujutsu's manpages to the provided path
- [`markdown-help`](./jj/util/markdown-help) — Print the CLI help for all subcommands in Markdown
