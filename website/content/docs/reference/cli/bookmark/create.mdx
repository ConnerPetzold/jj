Create a new bookmark

```shell title="Usage"
jj bookmark create [OPTIONS] <NAMES>...
```

**Command Alias:** `c`

### Arguments

- `<NAMES>` — The bookmarks to create

### Options

- `-r`, `--revision <REVSET>` [alias: `to`] — The bookmark's target revision
