Manage Git remotes

The Git repo will be a bare git repo stored inside the `.jj/` directory.

```shell title="Usage"
jj git remote <COMMAND>
```

### Subcommands

- [`add`](./jj/git/remote/add) — Add a Git remote
- [`list`](./jj/git/remote/list) — List Git remotes
- [`remove`](./jj/git/remote/remove) — Remove a Git remote and forget its bookmarks
- [`rename`](./jj/git/remote/rename) — Rename a Git remote
- [`set-url`](./jj/git/remote/set-url) — Set the URL of a Git remote
